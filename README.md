# Conceptual Metaphors in Mental Health Discourse

This repository contains a corpus used for the analysis of conceptual metaphors in mental health discourse.

The dataset includes testimonies from different mental health conditions, such as alcohol and drug-related issues, depression, bipolar disorder, obsessive-compulsive disorder, eating disorders, and psychosis.

## Files

The repository includes:

* the full corpus with metadata;
* the full corpus without metadata (only narratives);
* the subcorpus with metadata;
* the subcorpus without metadata (only narratives);
* the labelled subcorpus.

## Format

All files are provided in UTF-8 encoding. The datasets are mainly in TXT format, except the annotated subcorpus, which is provided in JSON format.

## Notes

The labelled subcorpus contains manual annotations related to conceptual metaphors and types of figurative language (metaphors, similes, and metonymies).
